Forte DAW へようこそ

    
このパッケージはまだ空です。曲は block(再利用できる数小節のかたまり)を並べて作ります。

    
    
      
  1. デモ曲から始めるのが早い — 開いて space で鳴ります
    2. 
      
  2. +block でグルーヴを作る(ビートグリッド / ピアノロールで打ち込み)
    3. 
      
  3. ライブラリの ▶ で block を単体試聴して育てる
    4. 
      
  4. +曲 で曲を作り、block の +曲 ボタンで配置していく
    5. 
      
  5. ミキサーとインスペクタで音を整える — 操作は全部コードに書き戻ります
    6. 
    

    

      
      
      
      
    

  





  操作ガイド

  space 再生 / 停止 · アレンジをダブルクリック そこへシーク

  クリップをドラッグ 配置を移動 · 右端ドラッグ 長さ変更

  ビートグリッド クリックで打ち込み · ロール ドラッグで音を描く / クリックで消す

  ミキサーのトラック名 インスペクタ(音源 / エフェクトのつまみ)

  M / S このセッションだけのミュート / ソロ

  左のツリー: +曲 / +block / +package · block の ▶ 試聴 / +曲 配置

  すべての GUI 操作は最小のコード編集として書き戻され、Ctrl+Z で戻せます
