  Gesture guide

  space play / stop · double-click the arrange to seek

  drag a clip to move a placement · drag its right edge to resize · right-click deletes · drop a block onto a clip swaps it

  beat grid click to program · roll drag to draw, click to delete, shift-click accent, alt-click tie

  a mixer strip's name opens the inspector (instrument / effect knobs, swap the instrument)

  M / S session-only mute / solo

  the tree: +song / +block / +package · a block's play button auditions, +song places it

  every GUI gesture is written back as a minimal code edit; Ctrl+Z (or ↶) undoes it
