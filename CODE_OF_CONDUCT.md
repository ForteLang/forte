# Code of Conduct

This project adopts the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

- Treat everyone with respect. Differences in experience, musical style, and
  technical background are assets to this project — the fork-and-lineage
  culture *is* the project.
- Harassment, personal attacks, and trolling are not tolerated.
- Respect the provenance of other people's work and recordings. Forging
  provenance or stripping credit is a community violation before it is a
  technical one.

## Reporting

If you see a violation, contact shusuke.inoue@fcuro.com.
Reports are handled confidentially.

## Scope

Applies to this repository (issues / PRs / discussions) and interactions on
the official Hub.
